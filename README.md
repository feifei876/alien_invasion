# 外星人入侵 🚀

用 Python 和 Pygame 写的经典游戏《外星人入侵》

## 🎮 游戏截图

（可以放截图）

## ✨ 游戏特性

- 三种难度选择（简单、普通、困难）
- 每种难度独立记录最高分
- 随着等级提升游戏速度加快
- 完整的音效系统
- 中文游戏说明

## 🎯 操作说明

| 按键 | 功能 |
|------|------|
| ← → | 移动飞船 |
| 空格键 | 发射子弹 |
| P | 开始游戏 |
| Q | 退出游戏 |
| ESC | 返回菜单 |
